# Markdown 语法完全指南

Markdown 是一种轻量级标记语言，让你可以使用纯文本格式编写文档。这篇文章将介绍 Markdown 的所有常用语法。

## 标题

使用 `#` 符号创建标题，数量表示标题级别：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 文本格式

### 强调

```markdown
*斜体文本* 或 _斜体文本_
**粗体文本** 或 __粗体文本__
***粗斜体文本*** 或 ___粗斜体文本___
~~删除线~~
```

效果：

*斜体文本*、**粗体文本**、***粗斜体文本***、~~删除线~~

### 引用

使用 `>` 创建引用：

```markdown
> 这是一段引用文本。
> 
> 可以有多行。
```

效果：

> 这是一段引用文本。
> 
> 可以有多行。

## 列表

### 无序列表

使用 `-`、`+` 或 `*` 创建无序列表：

```markdown
- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2
- 项目 3
```

效果：

- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2
- 项目 3

### 有序列表

使用数字加 `.` 创建有序列表：

```markdown
1. 第一项
2. 第二项
3. 第三项
```

效果：

1. 第一项
2. 第二项
3. 第三项

## 链接

### 内联链接

```markdown
[链接文本](https://example.com)
[带标题的链接](https://example.com "鼠标悬停显示的标题")
```

效果：[链接文本](https://example.com)

### 参考链接

```markdown
[链接文本][ref]

[ref]: https://example.com "参考链接"
```

## 图片

```markdown
![替代文本](image.jpg)
![带标题的图片](image.jpg "图片标题")
```

## 代码

### 内联代码

使用反引号包裹代码：

```markdown
这是 `内联代码` 示例
```

效果：这是 `内联代码` 示例

### 代码块

使用三个反引号创建代码块，可以指定语言以启用语法高亮：

````markdown
```javascript
function hello() {
  console.log("Hello, World!");
}
```
````

效果：

```javascript
function hello() {
  console.log("Hello, World!");
}
```

支持的语言包括：`javascript`、`python`、`java`、`html`、`css`、`bash` 等。

## 表格

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 数据1 | 数据2 | 数据3 |
| 数据4 | 数据5 | 数据6 |
```

效果：

| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 数据1 | 数据2 | 数据3 |
| 数据4 | 数据5 | 数据6 |

### 对齐

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
|:------|:-------:|------:|
| 左 | 中 | 右 |
```

## 分隔线

使用三个或更多的 `-`、`*` 或 `_` 创建分隔线：

```markdown
---
***
___
```

效果：

---

## 任务列表

```markdown
- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 另一个任务
```

效果：

- [x] 已完成的任务
- [ ] 未完成的任务
- [ ] 另一个任务

## 高级技巧

### 转义字符

使用反斜杠 `\` 转义特殊字符：

```markdown
\* 这不是斜体 \*
```

### 脚注

```markdown
这是一段包含脚注的文本[^1]。

[^1]: 这是脚注内容。
```

### HTML 标签

Markdown 支持内嵌 HTML 标签：

```html
<div style="color: red;">
  这是红色文本
</div>
```

## 实用工具

推荐的 Markdown 编辑器：

2. **VS Code** - 配合 Markdown 插件
3. **Obsidian** - 适合做笔记

## 总结

Markdown 语法简单易学，主要包括：

- 标题、文本格式、列表
- 链接和图片
- 代码和代码块
- 表格和分隔线

掌握这些语法，你就可以高效地编写格式优美的文档了！

> 💡 **小技巧**：多练习是掌握 Markdown 的关键，建议从现在开始用 Markdown 写笔记和文档。
